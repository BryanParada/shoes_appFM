# shoes_app

Check Professional design and animations of this shoe app

## Features  
- Choose shoe's size
- Change Color of the shoe

## What did i use in this app?

- Hero Animation
- Multiprovider
- Provider as state handler
- Many custom Widgets
- Import simplification
- Animate_do (a few tricks)
- Use of static images
